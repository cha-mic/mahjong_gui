# majong_gui
GUI for mahjong

## 目的
pythonのライブラリ"mahjong"をGUIで使う

## ルール
赤ドラ：あり <br>
喰いタン：あり <br>
副露：未実装 <br>

## 使い方
### 牌の入力
1. 入力したい場所の上のチェックボックスにチェックを入れる
    * チェックするボックスは1つだけ
    * チェックされているボックスが複数あるとエラーを吐く
1. 入力したい牌をクリック
    * ドラは**表示牌**をクリック
### オプションの入力
1. ツモ，立直などのオプションを入力
    * デフォルトではロンアガリ
### 計算
1. "計算"ボタンを押すと点数と役が表示される
    * 自風に"東"と入力していると親番の，それ以外は子の場合の点数が表示される
